# Проект: Место

## Использованные технологии
* HTML
* CSS
* Vanilla JS

## Поддерживаемые браузеры
* Chrome
* Edge
* Firefox ESR+
* Internet Explorer 10+
* Safari 8+
* Opera

## Инструкция по установке:
```
git clone https://github.com/hsh01/mesto.git
cd mesto
npm i webpack --save-dev
npm run build
npm run dev
```
